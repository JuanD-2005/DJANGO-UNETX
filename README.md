# 🐦 UNET - Better X (Clon de Twitter)

Proyecto desarrollado con **Django** que replica las funcionalidades básicas de Twitter:
- Registro y autenticación de usuarios.
- Creación, edición y eliminación de publicaciones.
- Seguimiento de usuarios (follow/unfollow).
- Sistema de menciones y mensajes privados.
- Edición de perfil con imagen y biografía.

---

## ⚙️ Requisitos

- Python 3.10 o superior  
- Django 5.x  
- Virtualenv (opcional pero recomendado)

---

## 🚀 Instalación y ejecución

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tuusuario/tu-repo.git
   cd tu-repo

2. Crea y activa un entorno virtual:
    ```bash
   python -m venv venv
   source venv/bin/activate
   ```

3. Instala dependencias:
   ```bash
   pip install Django
   ```
4. Aplica migraciones:
   ```bash
   python manage.py migrate
   ```
5. Inicia el servidor:
   ```bash
   python manage.py runserver
   ```

6. Abre tu navegador
  ```bash
   http://127.0.0.1:8000/
   ```
